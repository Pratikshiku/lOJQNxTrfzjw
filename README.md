### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 个人中心（修改头像、修改密码、修改个人信息）
4. 美食查看和搜索
5. 美食详情
6. 美食评论留言
7. 美食点赞点踩

**管理员功能：**
1. 登录
2. 注册
3. 个人中心
4. 修改密码
5. 用户管理
6. 美食鉴赏管理
7. 美食鉴赏评论留言管理
8. 我的收藏管理
9. 系统设置（轮播图设置）
